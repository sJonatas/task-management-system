# Task Management System

## Quick Start with Docker

```bash
# Clone repository
git clone https://github.com/sJonatas/task-management-system
cd task-management-system
```

# Start entire stack
```bash
docker-compose up -d
```

# Access application
* Frontend: http://localhost:3000
* Backend API: http://localhost:8000

Database: 

| Var Name | Value   |
|----------|---------|
| User     | user    |
| Password | passpwd |
| Port     | 3306    |
